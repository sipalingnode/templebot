# Install

## Install NodeJS

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs
```

## Clone Repository

```bash
git clone https://github.com/sipalingnode/templebot.git
cd templebot
```

## Install Dependencies

```bash
npm install
```

---

## Create Telegram Bot
* Open [BotFather](https://t.me/BotFather)
* Send: `/newbot`
* Enter bot name & username
* Copy the BOT TOKEN
## Get Telegram Chat ID
* Open [GetID](https://t.me/userinfobot)
* Send: /start
* Copy your Chat ID

# Add Account & Running Bot

**Jalankan bot**

```

```

**Lalu buka Telegram bot.**

**Command**

```
/startbot
/addaccount
```

**Isi data**

* **Nama akun**
* **API Key**
* **Loop Private Key**
* **Party ID**

# Telegram Commands

## Start Menu

```
/startbot
```

## Run Bot

```
/runbot
```

## Stop Bot

```
/stopbot
```

## Set Delay

```
/setdelay 30
```

## Change Mode

```
/mode buy
```

**atau**

```
/mode sell
```

## Check Temple Balance

```
/balance
```

## Check Loop Wallet

```
/wallet
```

## Deposit Manual

```
/deposit 1 100 cc
```

**atau**

```
/deposit 1 100 usdcx
```

## Check CC Price

```
/checkprice
```

## Check Gas Fee

```
/gasfee
```

## Detail Settings

```
/detailset
```
